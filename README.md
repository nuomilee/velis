# Velis（拾页）

一个更轻盈、更优雅的桌面整理盒子工具。

Velis（中文名暂定 **拾页**）是一个面向 Windows 桌面的整理工具：它会将桌面文件分类整理，并以悬浮盒子的形式直接展示在桌面上。你可以把它理解成一个“会整理桌面、也会把整理结果优雅地留在桌面上”的小工具。

它目前基于 **Python 3.10 + PySide6** 开发，支持规则整理与 AI 辅助改名 / 分类。项目仍在持续打磨中，已经能跑，但还远没有到“完美”的程度。

---

## 项目特色

- **桌面盒子展示**：将不同分类的文件以独立盒子悬浮显示在桌面上
- **AI 辅助整理**：支持用 AI 参与文件分类与重命名
- **规则模式兜底**：即使不启用 AI，也能完成基础的分类整理
- **支持快捷方式识别**：对 `.lnk` 文件做额外处理
- **支持拖动、缩放、吸附**：桌面盒子支持基础布局交互
- **正在向更精致的桌面体验演进**：包括模糊、预览图标、右键菜单、托盘体验等

## 安装

### 1. 创建虚拟环境

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

### 2. 安装依赖

```powershell
pip install -r requirements.txt
```

---

## 运行

```powershell
python run.py
```

如果你本机装有多个 Python，也可以这样：

```powershell
py -3.10 run.py
```

## 使用说明

### 基础流程
1. 打开软件
2. 配置 AI 模式（可选）
3. 点击“开始整理”
4. 桌面文件会被整理到统一目录
5. 分类结果会以桌面盒子的形式展示

### 当前交互（持续调整中）
- 盒子支持移动 / 缩放
- 支持隐藏盒子
- 支持通过托盘恢复显示盒子
- 支持不同展示样式切换

## 已知问题 / 后续改进方向

目前主要还有这些问题需要继续打磨：

1. 托盘图标响应过慢
2. 托盘菜单和盒子菜单很丑
3. 模糊效果没有圆角
4. 盒子内图片、音乐等文件没有预览图标
5. 盒子内文件没有右键文件菜单
6. GUI 界面很丑
7. AI 处理速度比较慢，需要并行处理
8. 日志显示格式详略不当，需要优化
9. 软件没有完整运行日志
10. AI 改名逻辑提示词需要优化

## 一些设计上的想法

Velis 并不只是想做一个“把文件挪到文件夹里”的整理脚本。

更想做的是一种介于桌面、文件夹、轻工作流之间的体验：
让桌面不只是堆文件，而是变成一种“可整理、可停留、可浏览”的空间。

所以这个项目现在虽然还有不少粗糙的地方，但方向很明确：
**不只是整理桌面，而是重新组织桌面的使用方式。**
