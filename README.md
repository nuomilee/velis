# Velis（拾叶）

一个更轻盈、更优雅的桌面整理盒子工具。

Velis（中文名 **拾叶**）是一个面向 Windows 桌面的整理工具：它会将桌面文件分类整理，并以悬浮盒子的形式直接展示在桌面上。你可以把它理解成一个会帮你整理桌面、又愿意把整理结果好好留在桌面上的小工具。

它目前基于 **Python 3.10 + PySide6** 开发，支持规则整理与 AI 辅助改名 / 分类。项目已经可以运行，也已经有了比较清晰的方向，不过离“真正顺手、真正好看、真正让人想一直用下去”还有一些距离。

---

## 它想做的事情

很多桌面整理工具只是在“把文件挪走”。

但 Velis 更想做的，是把桌面重新变成一个可以停留、浏览、整理、轻轻使用的空间。不是一股脑清空，也不是把东西塞进深层文件夹里，而是让整理后的内容仍然自然地留在桌面上，用一种更柔和、更直观的方式继续陪着你。

所以它不只是一个整理脚本，也不只是一个悬浮组件。
它更像是一个还在慢慢长大的桌面小工具。

---

## 现在已经有的能力

### 桌面文件整理
- 可以将桌面文件分类移动到统一目录中
- 支持规则模式整理
- 支持 AI 辅助分类与改名
- 对快捷方式（`.lnk`）有额外处理逻辑

### 桌面盒子展示
- 按分类生成独立的桌面悬浮盒子
- 支持基础拖动、缩放、吸附与隐藏
- 支持不同展示样式切换
- 支持通过托盘或盒子交互恢复显示

### 外观调整
- 背景透明度
- 背景颜色
- 选中颜色
- 选中透明度
- 模糊开关（还在继续优化）

## 安装

### 1. 创建虚拟环境

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

### 2. 安装依赖

```powershell
pip install -r requirements.txt
```

---

## 运行

```powershell
python run.py
```

如果你本机装有多个 Python，也可以这样：

```powershell
py -3.10 run.py
```

---

## 使用方式

### 基础流程
1. 打开软件
2. 配置 AI 模式（可选）
3. 点击“开始整理”
4. 桌面文件会被整理到统一目录
5. 分类结果会以桌面盒子的形式展示出来

### 当前交互（仍在调整中）
- 盒子支持移动 / 缩放
- 支持隐藏盒子
- 支持恢复显示盒子
- 支持不同展示样式切换

## 一起继续把它打磨好

这个项目现在已经有了一个比较清晰的骨架，但还有很多细节值得继续修、继续磨、继续变漂亮。

如果你也对桌面整理、悬浮组件、轻工作流工具、AI 文件整理这些方向感兴趣，欢迎一起把它往前推进。

目前最值得继续完善的方向包括：

1. 托盘图标响应过慢
2. 托盘菜单和盒子菜单视觉体验较差
3. 模糊效果与圆角不一致
4. 盒子内图片、音乐等文件缺少预览图标
5. 盒子内文件缺少更自然的右键文件菜单
6. GUI 界面整体仍需重做与美化
7. AI 处理速度偏慢，适合继续尝试并行处理
8. 日志显示格式详略不够合理
9. 软件缺少完整运行日志能力
10. AI 改名逻辑提示词仍需要优化

更详细的整理见：[`ROADMAP.md`](./ROADMAP.md)
